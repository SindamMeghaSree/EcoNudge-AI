# EcoNudge-AI
EcoNudge-AI is a smart campus sustainability dashboard built with Streamlit and Python. It tracks electricity and water usage across campus locations, generates real-time alerts, and computes a Campus Sustainability Score, CO2 Score, savings. The dashboard features dynamic graphs and charts to visualize trends and comparisons..


## 💡 Features
- **Real-time monitoring** of electricity and water usage across campus locations.
- **Interactive visualizations** using Altair for energy, water, and environmental data.
- **Alerts & notifications** for high usage, low resources, or unusual patterns.
- **Eco-friendly nudges** to encourage sustainable behavior.
- **Customizable dashboards** with different graphs, legends, and real-time updates.
- **Expandable insights** for detailed consumption analytics.
EcoNudge AI – Process Flow (How the Solution Works)
1. Data Ingestion

Electricity and water usage data is collected from campus sources (meters / CSV / IoT / simulated feed).

Data includes timestamp, location, electricity usage, and water consumption.

⬇️

2. Data Cleaning & Normalization

Invalid, missing, or inconsistent readings are removed.

Data is standardized into hourly and daily formats for analysis.

⬇️

3. Pattern Analysis

System analyzes historical and recent trends:

Peak vs non-peak hours

Normal vs abnormal usage

Location-wise consumption behavior

⬇️

4. Baseline Establishment

A normal usage baseline is calculated for each location.

This baseline represents expected consumption under healthy conditions.

⬇️

5. Intelligent Alert Detection

Current usage is continuously compared with baseline values.

Alerts are triggered when thresholds are crossed:

High electricity usage

Medium electricity usage

Water overuse

Water leakage

⬇️

6. Risk Prediction (Next 2 Hours)

Recent consumption trends are analyzed.

If usage is rising during known peak periods, the system predicts near-future risk.

Preventive alerts are generated before actual wastage occurs.

⬇️

7. Actionable Nudge Generation

Each alert is enriched with:

Location

Time

Type of issue

Recommended corrective action

⬇️

8. Visualization & Insights Dashboard

Alerts, trends, peaks, and comparisons are visualized in real time.

Users can quickly understand:

What is happening

Where it is happening

Why it matters

⬇️

9. Impact Measurement

User actions are translated into:

Estimated cost savings

CO₂ reduction

Campus sustainability score

⬇️

10. Decision & Intervention

Campus authorities take targeted action.

Reduced wastage feeds back into improved future baselines.
